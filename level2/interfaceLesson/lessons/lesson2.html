الاحتمال الشرطي

الاحتمال الشرطي ومسلماته:

إذا كان A,B حدثان يمكن تعريف الاحتمال الشرطي على أنه احتمال وقوع حدث معين A علماً أن حدثاً آخر B قد وقع فعلاً أي:

p ( A │ B ) = ( A ∩ B ) p ( B )

و هو احتمال وقوع الحدث B علماً أن الحدث A قد وقع:

p ( B │ A ) = p ( B ∩ A ) p ( A )

علماً أن p(A|B) ≠ p(B|A) إلا في حال كان p(A) = p(B)

بعض الملاحظات المتعلقة بالاحتمال الشرطي:

1]-

إذا كان الحدث A محتوى ضمن الحدث B أي: A⊂B وبالتالي: p(A∩B) = p(A) عندئذٍ: p(B│A) = p(A) / p(A) = 1

2]-

إذا كان الحدث B محتوى ضمن الحدث A أي: B⊂A وبالتالي: p(A∩B) = p(B) عندئذٍ: p(B│A) =p (B) / p(A)

3]-

إن احتمال وقوع الحدث المستحيل علماً أن أي حدث آخر قد وقع هو صفر أي:

p ( ∅ │ B ) = p ( ∅ ∩ B ) p ( B ) = p ( ∅ ) p ( B ) = 0
4]-

إذا كان احتمال التقاطع بين حدثين A,B معدوم فإن احتمال وقوع أحدهما بوقوع الآخر معدوم ايضاً أي:

p ( A ∩ B ) = 0 ⇒ p ( B │ A ) = 0

p ( A ∩ B ) = 0 ⇒ p ( A │ B ) = 0
5]-

إذا كان لدينا الحدث A والحدث B وكنا نريد حساب p(A|B) ولكن استعصى علينا حسابه وكان يمكننا حساب

p ( A ¯ | B )

فإننا يمكننا الاستفادة من القانون الآتي:

p ( A │ B ) = 1 − p ( A ¯ | B )
